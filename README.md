# CalculadoraDeMetricas
O objetivo é criar uma maneira de facilitar um aprendiz da disciplina de métrica de software, disponibilizando uma calculadora 
contendo como extra o cálculo de contagens indicativas, estimadas, bem como de outros elementos necessários como ALI, AIE etc.
